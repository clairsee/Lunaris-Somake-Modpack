# **Somake Lunáris Modpack**
Criado com a inteção de disponibilizar um Modpack Otimizado e com modificações feitas especificadamente para os membros da Lua

## Tópicos
* [Informações Gerais](#Informações-Gerais)
* [Objetivo](#Objetivo)

## Informações Gerais
Projeto feito por Mik, Lipe e Light para a inteção de oferecer a Lua:
* Modpack Otimizado
* FPS Maior
* Personalização Único
* Melhor Experiência dentro do Somake

## Objetivo
Nosso objetivo geral, como dito, é fornecer a todos os membros da Lua e outras pessoas interessadas no uso do mesmo, uma experiência personalizada e de acordo com o que eles precisam.
Utilizaremos de nosso conhecimento para fornecer tanto modificações em Modelos, Modificações, Texturas dentro do servidor do Somake.

>[!IMPORTANT]
>Tudo criado aqui será criado dentro das regras do [Somake SMP](https://www.instagram.com/somakesmp)

## Instalação
Iremos disponibilizar três maneiras de baixar a Personalização.
```
Disponibilizaremos o arquivo para uso no Modrinth
```
```
Disponibilizaremos o arquivo para uso no Curseforge
```
```
Baixando separadamente nos arquivos que estão disponíveis
```

>[!TIP]
>Recomendamos o uso da Terceira opção, por acreditarmos que você já possui um Modpack em uso. Assim, possibilidade a customização sem a mudança de Modpack inteiramente
